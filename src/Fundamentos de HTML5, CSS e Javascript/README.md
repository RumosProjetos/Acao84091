Aqui estão alguns exemplos de exercícios sobre formulários HTML:

1. **Formulário de Login Simples:**
   Crie um formulário de login com campos para nome de usuário e senha. Valide os campos para garantir que ambos sejam preenchidos antes de permitir o envio.

2. **Formulário de Registro de Usuário:**
   Desenvolva um formulário de registro de usuário com campos para nome, sobrenome, e-mail, senha e confirmação de senha. Adicione validações para garantir que o e-mail seja válido e que a senha tenha pelo menos 8 caracteres.

3. **Formulário de Contato:**
   Crie um formulário de contato com campos para nome, e-mail, assunto e mensagem. Adicione uma caixa de seleção para o tipo de consulta (por exemplo, suporte, feedback, etc.) e um botão de envio.

4. **Formulário de Pedido de Pizza:**
   Desenvolva um formulário para solicitar uma pizza personalizada com campos para tamanho, coberturas, nome e endereço de entrega. Utilize caixas de seleção, botões de rádio e campos de texto.

5. **Formulário de Pesquisa:**
   Crie um formulário de pesquisa com um campo de entrada de texto e um botão de envio. Adicione uma lista suspensa para permitir que os usuários selecionem uma categoria para a pesquisa, como "Tecnologia", "Esportes", "Política", etc.

6. **Formulário de Inscrição em Evento:**
   Desenvolva um formulário de inscrição em evento com campos para nome, e-mail, número de telefone e seleção de workshops opcionais. Adicione uma caixa de seleção para aceitar os termos e condições.

7. **Formulário de Avaliação de Produto:**
   Crie um formulário de avaliação de produto com campos para avaliação de estrelas, comentários e informações do cliente, como nome e e-mail. Valide os campos obrigatórios e limite os caracteres nos campos de comentários.

8. **Formulário de Envio de Arquivo:**
   Desenvolva um formulário que permita aos usuários enviar arquivos, como imagens ou documentos. Inclua um campo de entrada de arquivo e um botão de envio.

Esses são apenas alguns exemplos para começar. Você pode personalizá-los ou adicionar funcionalidades extras para tornar os exercícios mais desafiadores.

## Aula 03 - Conceitos fundamentais do CSS 

- [CSS Tutorial](https://www.w3schools.com/css/default.asp)


### Exercícios CSS
aqui estão alguns exemplos de exercícios sobre CSS que podem ajudar a praticar e entender melhor os conceitos:

1. **Seletor de classe e propriedades básicas:**
   - Crie uma classe chamada `.destaque` que altera a cor do texto para vermelho e aumenta o tamanho da fonte para 20px. Aplique essa classe a um parágrafo em seu HTML.

2. **Estilização de lista:**
   - Crie uma lista não ordenada (`<ul>`) e estilize os itens da lista para terem uma cor de fundo cinza claro e uma borda sólida de 1px em volta. Use pseudo-classes para estilizar os itens da lista ao passar o mouse sobre eles.

3. **Layout com Flexbox:**
   - Crie uma estrutura de layout básica usando Flexbox. Tenha um contêiner com três itens dentro. Faça com que esses itens se distribuam igualmente ao longo do contêiner e mantenham uma distância de 20px entre si.

4. **Responsividade com Media Queries:**
   - Crie uma página simples com algumas imagens e texto. Utilize media queries para garantir que o layout se ajuste adequadamente em dispositivos de tela pequena, como smartphones. Por exemplo, faça com que as imagens se redimensionem e o texto se ajuste para manter a legibilidade.

5. **Animação CSS:**
   - Crie uma animação simples onde um quadrado se move horizontalmente de um lado da tela para o outro. Use a propriedade `@keyframes` para definir a animação e a propriedade `animation` para aplicá-la ao elemento.

6. **Grade CSS (CSS Grid):**
   - Crie uma grade de 3x3 usando CSS Grid. Insira conteúdo nos itens da grade e estilize-os de forma que ocupem espaços iguais e tenham uma borda sólida de 1px.

7. **Estilização de formulários:**
   - Crie um formulário HTML básico com campos de entrada de texto, área de texto e botões. Estilize esses elementos usando CSS para melhorar a aparência e a usabilidade do formulário.

8. **Efeito de hover:**
   - Crie um efeito de hover em um elemento específico, como um botão ou um link. Por exemplo, faça com que a cor de fundo do elemento mude quando o mouse passar sobre ele.

Esses exercícios cobrem uma variedade de conceitos fundamentais de CSS e podem ser adaptados ou expandidos de acordo com as necessidades de aprendizado e prática.

### Exercícios JavaScript

Aqui estão 20 exercícios iniciais sobre JavaScript que cobrem conceitos básicos da linguagem:

1. **Variáveis e Operadores:**
```javascript
// Exercício 1: Declare duas variáveis, atribua valores a elas e faça a soma.
let a = 5;
let b = 10;
let soma = a + b;
console.log("A soma é:", soma);
```

2. **Condicionais:**
```javascript
// Exercício 2: Verifique se um número é par ou ímpar.
let numero = 7;
if (numero % 2 === 0) {
    console.log(numero + " é par.");
} else {
    console.log(numero + " é ímpar.");
}
```

3. **Laços (Loop):**
```javascript
// Exercício 3: Imprima os números de 1 a 5 usando um loop.
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
```

4. **Funções:**
```javascript
// Exercício 4: Crie uma função que retorne a soma de dois números.
function soma(a, b) {
    return a + b;
}
console.log("A soma é:", soma(3, 4));
```

5. **Arrays:**
```javascript
// Exercício 5: Declare um array e imprima cada elemento.
let frutas = ["maçã", "banana", "laranja"];
for (let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]);
}
```

6. **Métodos de Array:**
```javascript
// Exercício 6: Adicione um elemento a um array e remova outro.
let numeros = [1, 2, 3];
numeros.push(4); // Adiciona o número 4 ao array
numeros.pop();   // Remove o último elemento do array
console.log(numeros);
```

7. **Objetos:**
```javascript
// Exercício 7: Crie um objeto representando uma pessoa e imprima suas propriedades.
let pessoa = {
    nome: "João",
    idade: 30,
    profissao: "engenheiro"
};
console.log(pessoa.nome + " tem " + pessoa.idade + " anos e é " + pessoa.profissao + ".");
```

8. **Métodos de Objeto:**
```javascript
// Exercício 8: Adicione um método a um objeto e chame-o.
let carro = {
    marca: "Toyota",
    modelo: "Corolla",
    ano: 2020,
    descricao: function() {
        return this.marca + " " + this.modelo + " " + this.ano;
    }
};
console.log(carro.descricao());
```

9. **Manipulação de Strings:**
```javascript
// Exercício 9: Concatene duas strings.
let string1 = "Olá, ";
let string2 = "mundo!";
let mensagem = string1 + string2;
console.log(mensagem);
```

10. **Tratamento de Eventos (DOM):**
```javascript

    alert("O botão foi clicado!");

```

11. **Validar Campo Obrigatório:**
```html
<!--Crie um formulário com um campo de texto obrigatório. Escreva um script que valide se o campo está preenchido antes de permitir o envio do formulário. -->

    <input type="text" id="campoObrigatorio" required>
    <input type="submit" value="Enviar">
</form>

function validarForm() {
    let campo = document.getElementById("campoObrigatorio");

        return false;
    }
    return true;
}
</script>
```

12. **Validar E-mail:**
```html
<!--Crie um formulário com um campo de e-mail. Escreva um script que valide se o e-mail inserido é válido antes de permitir o envio do formulário. -->
<form id="meuFormulario" onsubmit="return validarForm()">
    <input type="email" id="campoEmail" required>
    <input type="submit" value="Enviar">
</form>

<script>
function validarForm() {
    let email = document.getElementById("campoEmail").value;
    let regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    if (!regex.test(email)) {
        alert("Por favor, insira um e-mail válido.");
        return false;
    }
    return true;
}
</script>
```

13. **Contador de Caracteres:**

14. **Seleção de Opções:**

15. **Checkbox:**
```html
<!--Crie um formulário com um checkbox e um botão. Escreva um script que valide se o checkbox está marcado antes de permitir o envio do formulário. -->
<form>
    <input type="checkbox" id="checkbox"> Concordo com os termos.
    <button onclick="validarCheckbox()">Enviar</button>
```

16. **Radio Buttons:**
```html
<!--Crie um formulário com radio buttons e um botão. Escreva um script que valide se um dos radio buttons está selecionado antes de permitir o envio do formulário. -->
<form>
    <input type="radio" name="opcao" value="opcao1"> Opção 1
    <input type="radio" name="opcao" value="opcao2"> Opção 2
    <input type="radio" name="opcao" value="opcao3"> Opção 3
    <button onclick="validarRadio()">Enviar</button>
</form>

<script>
function validarRadio() {
    let opcoes = document.getElementsByName("opcao");
    let selecionado = false;
    for (let i = 0; i < opcoes.length; i++) {
        if (opcoes[i].checked) {
            selecionado = true;
            break;
        }
    }
    if (!selecionado) {
        alert("Por favor, selecione uma opção.");
        return false;
    }
    return true;
}
</script>
```


17. **Máscara de Campo:**
```html
<!--Crie um campo de telefone que formate automaticamente o número inserido. Escreva um script que adicione parênteses, traço e espaços ao número à medida que o usuário digita. -->
<form>
    <input type="text" id="campoTelefone" oninput="formatarTelefone()">
</form>

<script>
function formatarTelefone() {
    let campo = document.getElementById("campoTelefone");
    let valor = campo.value.replace(/\D/g, "");
    let regex = /^(\d{3})(\d{4,5})(\d{4})$/;
    valor = valor.replace(regex, "($1) $2-$3");
    campo.value = valor;
}
</script>
```

18. **Upload de Arquivos:**
