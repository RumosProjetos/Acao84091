😀 Exercício 02 - Responsividade

Modifique o layout criado no exercício 1 para que seja totalmente responsivo, ou seja, deve se ajustar bem em telas de diferentes tamanhos, desde smartphones até desktops.

Para tornar o layout totalmente responsivo, podemos usar as classes de grid do Bootstrap para garantir que os elementos se ajustem automaticamente em telas de diferentes tamanhos. Aqui está o código HTML atualizado:

respostas/resposta02.html

Neste exemplo, usei a classe container para envolver todo o conteúdo e a classe container dentro das seções para garantir que o conteúdo permaneça centralizado e responsivo. Para a seção principal, usei a classe col-lg-8 offset-lg-2 para garantir que o conteúdo seja exibido em uma largura de 8 colunas no layout grande (desktop) e centralizado horizontalmente. O offset-lg-2 cria um espaço em branco equivalente a 2 colunas no lado esquerdo. Isso garante que o conteúdo fique alinhado com a barra de navegação.

😀 Exercício 03 - Grid System:

Crie uma grade de layout usando o sistema de grid do Bootstrap v5. Divida a página em pelo menos 3 colunas e faça com que essas colunas sejam responsivas.

Aqui está um exemplo de como você pode criar uma grade de layout responsiva dividindo a página em pelo menos 3 colunas usando o sistema de grid do Bootstrap v5:

respostas/resposta03.html

Neste exemplo:

  • Usei a classe container para envolver o conteúdo e garantir que as colunas estejam alinhadas corretamente.
  • Dentro da classe container, criei uma row para conter as colunas.
  • Cada coluna é definida com a classe col-md-4 , o que significa que em dispositivos médios (como tablets), cada coluna ocupará 4 das 12 colunas disponíveis (ou seja, 1/3 da largura total).
  • Usei classes de cores de fundo diferentes (bg-primary, bg-secondary, bg-info) para destacar cada coluna, tornando mais fácil distinguir uma da outra.
  • Dentro de cada coluna, adicionei algum conteúdo fictício para ilustrar como você pode preencher cada seção.
