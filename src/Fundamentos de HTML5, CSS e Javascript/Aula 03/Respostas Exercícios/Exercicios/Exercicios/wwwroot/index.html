Exercícios

Esses exercícios cobrem uma variedade de conceitos fundamentais de CSS e podem ser adaptados ou expandidos de acordo com as necessidades de aprendizado e prática.

  1. Crie uma classe chamada .destaque que altera a cor do texto para vermelho e aumenta o tamanho da fonte para 20px. Aplique essa classe a um parágrafo em seu HTML. Estilização de lista: Respostas/Ex01.html
  2. Crie uma lista não ordenada e estilize os itens da lista para terem uma cor de fundo cinza claro e uma borda sólida de 1px em volta. Use pseudo-classes para estilizar os itens da lista ao passar o mouse sobre eles. Layout com Flexbox: Respostas/Ex02.html
  3. Crie uma estrutura de layout básica usando Flexbox. Tenha um contêiner com três itens dentro. Faça com que esses itens se distribuam igualmente ao longo do contêiner e mantenham uma distância de 20px entre si. Responsividade com Media Queries: Respostas/Ex03.html
  4. Crie uma página simples com algumas imagens e texto. Utilize media queries para garantir que o layout se ajuste adequadamente em dispositivos de tela pequena, como smartphones. Por exemplo, faça com que as imagens se redimensionem e o texto se ajuste para manter a legibilidade. Animação CSS: Respostas/Ex04.html
  5. Crie uma animação simples onde um quadrado se move horizontalmente de um lado da tela para o outro. Use a propriedade @keyframes para definir a animação e a propriedade animation para aplicá-la ao elemento. Grade CSS (CSS Grid): Respostas/Ex05.html
  6. Crie uma grade de 3x3 usando CSS Grid. Insira conteúdo nos itens da grade e estilize-os de forma que ocupem espaços iguais e tenham uma borda sólida de 1px. Estilização de formulários: Respostas/Ex06.html
  7. Crie um formulário HTML básico com campos de entrada de texto, área de texto e botões. Estilize esses elementos usando CSS para melhorar a aparência e a usabilidade do formulário. Efeito de hover: Respostas/Ex07.html
  8. Crie um efeito de hover em um elemento específico, como um botão ou um link. Por exemplo, faça com que a cor de fundo do elemento mude quando o mouse passar sobre ele. Respostas/Ex08.html
