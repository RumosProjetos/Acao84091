Crie um formulário HTML básico com campos de entrada de texto, área de texto e botões. Estilize esses elementos usando CSS para melhorar a aparência e a usabilidade do formulário.

Formulário de Contato

Neste exemplo:

  • O formulário está contido em um [div] com a classe .form-container, que define a largura máxima, o preenchimento, a cor de fundo, o raio de borda e a sombra para melhorar a aparência.
  • Os campos de entrada de texto e a área de texto têm largura de 100%, preenchimento interno, margem, borda sólida, borda arredondada e tamanho de caixa de conteúdo para uma aparência uniforme.
  • O botão de envio ([button]) tem largura de 100%, preenchimento interno, cor de fundo, cor do texto, borda sem estilo, borda arredondada e cursor de apontador. Além disso, tem uma regra de estilo de hover para alterar a cor de fundo quando o mouse passa por cima.
